# KLP_CSH_Polymer_Stone_PriorArt
Low-Carbon C-S-H Polymer Stone (KLP) Process: Ambient-cure, non-calcination method for structural material achieving 7 Mohs hardness. Uses supercooled Na2SiO3 slurry and Ca(OH)2 activator. Royalty-free CC BY disclosure to establish Prior Art and promote global sustainable construction.
